<h1>Overordnet liste over selvantændelige materialer og stoffer på et værksted</h1>

<p>
De mest almindelige reelle selvantændelsesrisici i træværksteder er:
</p>

<ol>
  <li>Olieholdige klude, for eksempel med linolie</li>
  <li>Savsmuld med olie eller lak</li>
  <li>Støvopsamlingsfiltre</li>
  <li>Hærdende olieprodukter</li>
  <li>PU skum under hærdning</li>
</ol>

<p>
Disse står for en stor del af værkstedsbrande relateret til selvopvarmning.
</p>

<p>
I et almindeligt værksted med <strong>træ, skum og lidt metalbearbejdning</strong> findes der flere materialer,
som kan <strong>selvantænde</strong> eller øge risikoen for selvantændelse.
Mekanismerne er typisk:
</p>

<ul>
  <li>Oxidation som udvikler varme</li>
  <li>Varmeophobning i porøse materialer</li>
  <li>Kemiske reaktioner i hærdende produkter</li>
</ul>

<p>Her er en <strong>overordnet liste over relevante materialetyper</strong>.</p>

<h2>1. Olieholdige klude og absorberende materialer</h2>

<p>Den mest almindelige årsag til selvantændelse i træværksteder.</p>

<p><strong>Eksempler</strong></p>
<ul>
  <li>Klude med linolie</li>
  <li>Klude med tungolie</li>
  <li>Klude med teakolie</li>
  <li>Klude med hærdende træolier</li>
  <li>Papir eller savsmuld der har absorberet disse olier</li>
</ul>

<p><strong>Hvorfor</strong></p>
<p>
Hærdende olier oxiderer i luft. Oxidationen udvikler varme.
I en sammenkrøllet klud kan varmen ikke slippe væk, så temperaturen stiger og kan føre til antændelse.
</p>

<h2>2. Savsmuld og træstøv</h2>

<p>Især hvis det er:</p>
<ul>
  <li>meget fint</li>
  <li>fugtigt</li>
  <li>blandet med olie eller lakrester</li>
</ul>

<p><strong>Risiko</strong></p>
<ul>
  <li>langsom oxidation</li>
  <li>varmeophobning i bunker eller støvfiltre</li>
  <li>støveksplosioner hvis det hvirvles op</li>
</ul>

<h2>3. Polyurethanskum og skumrester</h2>

<p>Typisk fra:</p>
<ul>
  <li>PU byggeskum</li>
  <li>PU støbeskum</li>
  <li>visse isoleringsskum</li>
</ul>

<p><strong>Risiko</strong></p>
<ul>
  <li>under hærdning udvikles varme</li>
  <li>store klumper eller affald kan holde på varmen</li>
  <li>nogle komponenter kan reagere eksotermt hvis de blandes forkert</li>
</ul>

<h2>4. Oliebaserede lakker, fernis og bejdser</h2>

<p>Eksempler:</p>
<ul>
  <li>alkydlak</li>
  <li>oliebaseret træbeskyttelse</li>
  <li>oliebejdse</li>
</ul>

<p><strong>Risiko</strong></p>
<p>
Samme mekanisme som linolie. Problemet opstår især i klude, pensler, filtre og savsmuld.
</p>

<h2>5. Filtre og støvposer fra udsugning</h2>

<p>Støvopsamlere kan indeholde:</p>
<ul>
  <li>træstøv</li>
  <li>olieholdigt støv</li>
  <li>lakrester</li>
</ul>

<p>
Hvis de fyldes og bliver varme, kan oxidation starte inde i filteret.
</p>

<h2>6. Batterier, især lithium</h2>

<p>Findes i:</p>
<ul>
  <li>batteridrevet værktøj</li>
  <li>batteripakker</li>
  <li>gamle batterier i affald</li>
</ul>

<p><strong>Risiko</strong></p>
<ul>
  <li>kortslutning</li>
  <li>mekanisk skade</li>
  <li>termisk runaway, altså ukontrolleret selvopvarmning</li>
</ul>

<p>
Dette er ikke klassisk selvantændelse, men det kan starte brand uden ekstern flamme.
</p>

<h2>7. Metalpulver og meget fint metalstøv</h2>

<p>Mindre almindeligt i træværksteder, men kan forekomme ved slibning.</p>

<p>Eksempler:</p>
<ul>
  <li>aluminiumstøv</li>
  <li>magnesiumstøv</li>
</ul>

<p><strong>Risiko</strong></p>
<ul>
  <li>kan selvantænde i meget fin form</li>
  <li>kan reagere voldsomt hvis det først brænder</li>
</ul>

<h2>8. Kemikalier til hærdning eller limning</h2>

<p>Visse systemer udvikler varme under reaktion.</p>

<p>Eksempler:</p>
<ul>
  <li>epoxy med hærder</li>
  <li>polyesterharpiks</li>
  <li>PU limsystemer</li>
</ul>

<p><strong>Risiko</strong></p>
<p>
Store mængder i en beholder kan blive meget varme under hærdning.
</p>
